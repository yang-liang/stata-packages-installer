# ECON 640: Econometrics I

This repository contains teaching materials for ECON 640 (Graduate Econometrics), part I of the SDSU MS Economics program core sequence.

## Repository Structure

- **homework-solutions/**: Contains solutions to homework assignments
- **lecture-notes/**: Contains lecture notes and course materials
- **slides/**: Contains lecture slides
- **colab/**: Contains codes for teamwork

## Course Information

- **Course**: ECON 640 - Econometrics I for SDSU MS Economics Core Sequence

## Usage

These materials are organized to support the teaching and learning of econometric methods at the graduate level.
